# 朗読動画作成アプリ

音声ファイルとテキスト字幕、背景画像、BGMを直感的に組み合わせ、字幕演出を伴う朗読動画を自動生成するPySide6（Qt）製のデスクトップアプリケーションです。

---

## 🌟 主な機能

### 1. タイムライン＆クリップ管理

- **ドラッグ＆ドロップ並び替え**: 音声クリップ一覧の順番をドラッグ操作で直感的に変更可能です。並び替え後は自動でIDがリナンバリングされます。
- **音声の自動正規化コピー**: 追加された音声ファイルは、自動的に標準WAV形式（48kHz / 16bit PCM / ステレオ）へ変換・コピーされるため、形式違いによるエラーを防ぎます。
- **個別カスタマイズ**: 各音声クリップに対し、個別に「字幕テキスト」「次のクリップまでの無音余白（秒）」「個別音量倍率（最大2.0倍）」「有効/無効化」を設定できます。

### 2. 背景画像・BGM/環境音のタイムラインスパン設定

- **スパン指定システム**: アセット（背景画像、BGM）の開始タイミング・終了タイミングを「開始クリップID」から「終了クリップID」の範囲で感覚的に指定できます。
- **音量倍率の拡張**: 音声クリップと同様に、BGMアセットに対しても最大 **2.0倍（200%）** までの音量増幅設定が可能です。
- **リアルタイムフェード試聴**: BGMの「試聴」ボタンを押すと、設定されたフェードイン・フェードアウト秒数に基づいてリアルタイムに音量が線形フェードする状態でアセットのサンプル視聴が可能です。
- **ループ・有効化制御**: 各アセットごとに有効/無効化や、BGMのループ再生フラグを個別にオン・オフできます。

### 3. 高機能プレビューとスタイリング

- **動的プレビュー切り替え**:
  - **音声クリップ選択時**: 背景画像に字幕がオーバーレイされた、完成動画と同様のプレビューがリアルタイム合成されます。
  - **背景アセット選択時**: 字幕を含まない背景画像単体（アスペクト比維持のフルカバー）のプレビューに動的に切り替わります。
- **アプリ・プロジェクト設定**:
  - 字幕フォント（システムフォントからの自動探索 ＋ プロジェクトフォルダ内 `fonts/` ディレクトリからの最優先ロードに対応）
  - 字幕サイズ、位置（上・中・下）、文字色、背景不透明黒ボックスの不透明度、下部マージン
  - デフォルト無音余白の秒数

### 4. 堅牢な設計と動画生成

- **非同期レンダリング**: MoviePy・FFmpegを用いた重いレンダリング処理はバックグラウンドの別スレッド（`RenderWorker`）で実行されるため、生成中もGUIがフリーズしません。
- **未使用アセットクリーンアップ**: プロジェクトフォルダ内から、使用されなくなった音声や画像ファイルを安全に一括削除する機能を備えています。
- **Python 3.13完全対応**: アプリ終了時のスレッド・タイマー競合バグを回避する安全な終了シークエンス（`os._exit` によるクローズ）を搭載しています。

## 📂 プロジェクトのフォルダ構成

プロジェクトを開く、または新規作成すると、以下の構成でデータが自動管理されます。

```
[プロジェクトフォルダ]
│  project.json        # プロジェクトの設定・アセット割り当て情報（自動保存）
│
├─audio/               # 登録された音声ファイル（自動変換後の標準WAV）
│      clip_0001.wav
│      clip_0002.wav
│
├─images/              # 登録された背景画像ファイル
│      bg_0001.png
│
└─bgm/                 # 登録されたBGMファイル
       bgm_0001.mp3
```

## 🚀 動作環境・セットアップ

- Python 3.8 〜 3.13

### 1. 依存ライブラリのインストール

以下のコマンドで、GUIおよび動画生成に必要なライブラリ（PySide6, MoviePy, Pillow, imageio 等）を一括インストールします。

```bash
pip install -r requirements.txt
```

### 2. アプリの起動

アプリのルートディレクトリで以下を実行して起動します。

```powershell
python gui.py
```

## 🛠️ 配布用パッケージング（EXE化）

Python環境が入っていないPCでも動作する単一の実行可能ファイル（Windows向けEXE）を作成するには、`PyInstaller` を使用します。

### 1. PyInstallerのインストール

```bash
pip install pyinstaller
```

### 2. ビルドコマンドの実行

プロジェクトのルートディレクトリで以下のコマンドを実行します。

```powershell
pyinstaller --onedir --noconsole --name "朗読動画作成アプリ" --add-data "core.py;." --add-data "renderer.py;." gui.py
```

- ビルド完了後、`dist/朗読動画作成アプリ` フォルダが生成されます。
- 配布する際は、このフォルダごとZIPなどに圧縮して配布してください。
- ユーザーにカスタムフォントを使用させたい場合は、上記ビルドフォルダ（`朗読動画作成アプリ` フォルダ）の中に `fonts` フォルダを作成し、その中にフォントファイルを配置するように案内してください。
