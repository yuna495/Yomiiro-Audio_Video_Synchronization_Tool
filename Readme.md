# 朗読動画作成アプリ

音声ファイルとテキスト字幕、背景画像、BGMを統合し、美しい演出効果を伴う朗読動画を自動生成するデスクトップアプリケーションです。

## 主な機能

- **タイムライン＆クリップ管理方式**
  - 特定のファイル名（001/002等）の前提を廃止し、任意の音声ファイルを追加して自由な順序に並び替えることができます。
  - 音声を追加する際、自動で標準WAV（48kHz/PCM 16bit/stereo）へ正規化コピーされるため、音質や形式の不一致によるエラーが発生しません。
- **直感的な操作UI (PySide6 採用)**
  - リスト内の音声クリップを**ドラッグ＆ドロップで直接並び替える**ことができます。
  - 画面の仕切り幅をドラッグで調整できる `QSplitter` を採用し、ユーザーの好みのレイアウト幅に調整できます。
  - ダークテーマに調和した美しくフラットなマテリアル風デザインを適用しています。
- **リアルタイム静止画プレビュー**
  - 選択中の音声クリップに対して、指定した背景画像と字幕スタイル（文字サイズ、位置、色、背景ボックス不透明度、マージン）がどのように描画されるかを、中央のプレビュー画面でリアルタイムに確認できます。
- **背景画像＆BGMの範囲指定**
  - 各アセットの表示・再生タイミングを「開始クリップID」から「終了クリップID」の範囲で感覚的に指定できます。
  - 背景画像のクロスフェード切り替えや、BGMの自動フェードイン/フェードアウト、ループ再生にも対応しています。
- **クリーンアップ機能**
  - プロジェクト内で使用されなくなった音声や画像ファイルをフォルダから一括クリーンアップする機能を搭載しています。

## プロジェクトフォルダ構成の例

アプリで新規プロジェクトを作成、またはロードした際、以下の構成でデータが自動管理されます。手動でフォルダを作る必要はありません。

```
[プロジェクトフォルダ]
│  project.json        # プロジェクト設定（自動保存・ロード用）
│
├─audio/               # 登録された音声ファイル（自動変換・コピー保存）
│      clip_0001.wav
│      clip_0002.wav
│
├─images/              # 登録された背景画像（自動コピー保存）
│      bg_0001.png
│
└─bgm/                 # 登録されたBGMファイル（自動コピー保存）
       bgm_0001.mp3
```

## 動作環境・必要ライブラリ

本アプリの動作には Python 3.8 以上が必要です。

### 依存パッケージのインストール

以下のコマンドを実行して、必要なパッケージを一括インストールしてください（PySide6 が含まれています）。

```bash
pip install -r requirements.txt
```

### フォントの設定

アプリで字幕を描画する際に使用する日本語フォントについて：
- **デフォルトの動作**: Windows 標準のフォント（游明朝、MS明朝、MSゴシックなど）が自動的に探索されて使用されます。
- **カスタムフォントの使用**: 任意の日本語フォント（`.ttf`, `.otf`, `.ttc` 形式）を使用したい場合は、アプリの実行ディレクトリ（`gui.py` と同じフォルダ）直下、またはその配下に `fonts` という名前のフォルダを作成し、そこにフォントファイルを配置してください。自動的に最優先で読み込まれます。

## 起動方法

アプリのディレクトリに移動し、以下のコマンドで起動します。

```powershell
python gui.py
```

## 配布用パッケージング（実行ファイルの作成）

PyInstaller を使用することで、Python 環境のない PC でも動作する単一の実行可能ファイル（EXEファイル）を作成できます。

### 1. PyInstaller のインストール
```bash
pip install pyinstaller
```

### 2. 実行ファイルのビルド
アプリのルートディレクトリで以下のコマンドを実行します。
```powershell
pyinstaller --onedir --noconsole --name "朗読動画作成アプリ" --add-data "core.py;." --add-data "renderer.py;." gui.py
```
※ ビルド完了後、`dist/朗読動画作成アプリ` フォルダ内に実行可能ファイルが生成されます。配布時はこのフォルダごとZIP等に圧縮して配布してください。任意のフォントを使用させたい場合は、ユーザーがそのフォルダ内にフォントファイルを配置できるように案内してください。
